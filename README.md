# Hero Video - Modern Next.js App

A modern, responsive Next.js application featuring a stunning hero video section with overlay text and smooth animations.

## Features

- ✨ Modern, clean design with human-centered UI/UX
- 🎥 Auto-playing hero video background with controls
- 📱 Fully responsive navbar with mobile menu
- 🎨 Beautiful gradient overlays and glass morphism effects
- ⚡ Smooth animations and micro-interactions
- 🔧 Built with TypeScript and Tailwind CSS
- 🎯 Accessibility-focused design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Language:** TypeScript

## Design Features

- **Glass morphism effects** for modern UI elements
- **Gradient overlays** for better text readability
- **Smooth scroll indicators** and animations
- **Interactive video controls** (play/pause, mute/unmute)
- **Responsive design** that works on all devices
- **Professional typography** with Inter font family

## Customization

- Replace the video source in `app/page.tsx` with your own video
- Modify colors and gradients in `tailwind.config.js`
- Update content and copy in the hero section
- Add your own branding in the navbar

Enjoy your modern hero video website! 🚀
